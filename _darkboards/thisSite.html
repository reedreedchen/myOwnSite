Update Log

Although this website appears to be a portfolio site, it is actually a little crafting project/hobby of mine. I spent a sleepless night making the main layout in 2013. Since then, I have used it to demenstrate many of my experiments for web development. There were limited records before I decided to document this site and created this particular page in 2016.

Jan 2017

  • Contributed this website on Github.
  • Organized scripts developed for the Home Healthcare project on Github.
  • Gallery update:
    • Replaced the Github text with Github logo so the viewers can read less texts.
    • Added preview images to 'Design - Other' and 'Storyboards.'
    • Added preview images to '3D creation.'
    • When clicking on the animation link at the bottom, page scrolls back to top.
  • Found that Firebase and Github don't support I.E. very well. However, I found that Github works fine through RawGit. Since RawGit doesn't have 100% uptime, only I.E. users will be directed there.

Dec 2016

  • The domain name is trasferred to Google Domains.
  • Added the translation function by manually translating this website and by using Javascript. Users can choose to view this site in Traditional Chinese without being redirected to another page. The variable for language control will be stored in the client's local memory to be kept consistent because Javascript does not allow static variables.
